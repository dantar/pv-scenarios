Wanna play a game? Here's how!

This game takes place in a few real locations. The locations are shown on an interactive map. You'll see it when you close this window.

Every location has a game you should solve. When you tap a location you are provided with the instructions on how to pass that game. Every game assumes that you go to that location to find out the clues that are necessary to pass the game.

Every game is passed by providing a number code. Just type in the code and press the "GO!" button to see if the game is passed or not.

There is no penalty for trying out wrong codes. When you press GO with the right number code, the game is passed. Otherwise you'll be asked to try again.

You must pass all games in all locations in order to complete the whole game!

On the map there are other icons.

This one in the upper left is used to bring up these instructions again. Get back to these instructions as often as you need.

These buttons in the lower right allow you to center the map at:

  • all the game places
  • all the game places including your position
  • just your position

In case your position is needed on the map, your device will be asked whether you want to allow the application to use your GPS. Please allow it to use it so that you can see your position on the map.

Good luck!
